Flavours in Java
================
We have three flavours in Java.

> Standalone App 
-----------------
A normal java program which contains main method is called standlone application.

ex:
	class Test
	{
		public static void main(String[] args)
		{
			-
			- // some logic
			-
		}	
	}

> Desktop App 
--------------
It is a software application specially designed to perform perticular task.

ex:
	Control Panel 
	VLC Media player 
	Recycle Bin 


> Two-tier App 
---------------
Having more then one tier is called two tier application.

Diagram: class1.2


> Distributed App 
-----------------
Diagram : class1.3

In client/server application, if multiple request goes to main server then it will distribute those request to it's parallel servers to reduce the burdun of main server such type of application is called distributed application.


> Enterprise App
-----------------
An application which deals with large business complex logic with the help of middleware services 
is called enterprise application.

Here middleware services means authentication, autherization, malware production, security and etc.

ex:
	Facebook
	Online shopping websites 


> ERP App 
----------
ERP stands for Enterprise Resource Planning.

It is used to maintain the data in a enterprise.

Diagram: class1.4


> N-tier App 
------------
Having more then two tiers is called N-tier application.


> Mobile App 
---------------
It is a software application or a program which is designed for wireless network devices like phone, cell, tab, cellular and etc. rather then laptop's and pc's.

ex:
	pubG 
	Gpay 
	templaterun
	and etc.
